# HTML-website-styled-CSS
Styling a website using CSS

## Use of CSS
1. In this project we have styled our basic website using CSS
2. The website is responsive as properties like media queries and flex box have been used in styling
3. Very basic styling is also given to the contact page
4. A color scheme has been followed and google fonts has been used to enchnace typography

## How to run the project
1. Clone the given repository on your local device 
2. Open the index.html page in your default browser
